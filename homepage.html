Welcome to the (non-official) pokemon index!

This website shows you ALL the pokemon to ever exist and all of their stats, organized by their type.

How was this index made?

This website used a .csv that included pokemon information scraped from the official website. Python was used to parse the data and automate the creation of 800+ pages for each individual pokemon. All required files compiled into one HTML page.
